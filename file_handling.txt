Program 2: Count words, lines, characters
Python


Run
f=open("sample.txt","r")

data=f.read()

characters=len(data)

words=len(data.split())

f.seek(0)

lines=len(f.readlines())

print("Characters:",characters)
print("Words:",words)
print("Lines:",lines)

f.close()
Program 3: Copy one file to another
Python


Run
f1=open("sample.txt","r")
